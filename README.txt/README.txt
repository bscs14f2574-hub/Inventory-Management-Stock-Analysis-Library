PROJECT DESCRIPTION:
--------------------
This project is a custom C++ Library designed to manage retail inventory and 
analyze stock levels. It demonstrates core OOP concepts through a real-world 
problem model of a warehouse or store management system.

TECHNICAL FEATURES:
-------------------
1. Function Overloading: 
   - Multiple constructors for different ways to initialize products.
   - Overloaded addProduct functions in the Inventory class.

2. Operator Overloading:
   - '+' Operator: Merges stock of two products with the same ID.
   - '-' Operator: Handles sales by reducing stock quantity.
   - '==' Operator: Compares two products based on their unique ID.
   - '<<' Operator: Custom display format for product details.

3. Stock Analysis:
   - Calculates total inventory value.
   - Identifies the most expensive product in stock.
   - Low stock alerts (Visual indicators).

FILES INCLUDED:
---------------
1. YourLibrary.h: Contains the Product and Inventory class logic.
2. main.cpp: Interactive menu-driven program to test all features.
3. README.txt: Project documentation.
